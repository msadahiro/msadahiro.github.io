What's For Lunch

A website to vote on where to eat

Situation

    There have been countless times where nobody can decide where to eat or they nobody wants to suggest something.

Task

    To develop a voting app centered around voting for places to eat. The website needed user sign in ability, input fields for restaurants, and the ability to vote.

Action

    I owned every step of the product from building the front-end in React and connecting it to the backend in Firebase.

Results

    The total development time of the project was about two months. The application allows for users to sign in and vote on where to eat for lunch.

Technologies used

    HTML5 | CSS3 | JavaScript | ReactJS | Firebase | Git
