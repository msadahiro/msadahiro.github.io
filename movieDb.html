My own Movie Db

A website that suggests movie recommendations based on movies rated

Situation

    I am an avid movie watcher. I watch about 2 movies per week. I did not find a movie recommendation website that I liked, so I decided that I'd make my own.

Task

    To build a movie recommendation site based on movies rated. The home page needs a header explaining what the website is intended to provide. Needs a section for movie recommendations where movies are dynamically displayed based on what the user has rated. In addition, it needs a section for previously rated movies. The second page needs to be able to search for titles and be able to rate them. Once rated, they will remain persistant through the session and populate the main page.

Action

    I owned every step of the product from building the front-end in HTML, CSS, and JavaScript. I connecting it to an Express and Node.JS back-end storing rated movies into Firebase.

    I chose not to use any particular framework/library because I wanted to hone my JavaScript skills.

Results

    The total development time of the project was about two months. The application allows for users search for movies and rate them. When the user goes back to the Home page, they will see movies that have been previously reviewed and it will suggest movies to watch based on genre.

Technologies used

    HTML5 | CSS3 | JavaScript | NodeJS | Express | Firebase | GitHub | Heroku
