Tx Genetics

A self-help website

Situation

    Tx Genetics is a nutritional supplement company that has created a product that can help individuals livings with Interstitial Cystitis (permanent urinary tract infection)to manage and nearly eliminate their pain.

    The client needed an informational website with user accounts. The website's primary focus would be to educate others about the condition as well as provide hope to others suffering from Interstitial Cystitis. Vistors that experience these symptoms could sign up to participate in the trial by creating an account and keep of their progress while going through the trial.

Task

    My task was to help users understand what Tx Genetics is and how it can help them.

    In addition, to create a user portal to participants to navigate through an easy-to-use dashboard to keep track of progress throughout the trial.

    Lastly, I created an administrative login to allow administrators to keep track of participants progress and communicate with them.

Action

    I personally owned the user dashboard implementation with successful blog submission posts to the back-end. Also, was a key driver in implementing new features to increase user experience.

Results

    Product still in production. Will update upon completion.

Technologies used

    HTML5 | CSS3 | JavaScript | ReactJS | Redux | NodeJS | AWS-Serverless | Github
